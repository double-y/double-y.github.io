The Importance of Data Analysis

November 25, 2015

My major is data science. It is a field which deals with large amount of data and which finds some useful indications. It is getting more and more important.

A lot of people use at least excel in their job. It is a basic way to analyze data. Actually, if you ignore the limitation of computers, excel is enough to deal with all data. But it has two limitations.

1, It has limitation of the number of data. It uses a lot of computer resource to make itself easy to use so we need more efficient software like database. Database is a very efficient software to deal with data but you need some knowledge to use it. But if you learn about database even a little, you can understand the efficient way to organize data.

2, It is not efficient to process complex data. To find good indicator, it is often necessary to calculate. It’s very difficult to do it with basic functions in excel. Programming is better for such process because it is a tool created to make calculation efficient.

So we usually use database and programming but essentially, it’s not so different from excel. You should not think that you are not related to it or you don’t need it for your life if you have used excel.

There are two reasons why it is getting more and more important.

1, Many devices are generating large amount of data. Almost everyone has at least one computer including smartphone. Many electric devices like refrigerators are also starting to be a computer.

2, It’s almost inevitable to think about data in order to make decisions because the society is getting too complex. Most options have usually pros and cons. I often hear discussions between people who agree on it because of its merit and people who is against it because of its demerit. The only way to solve this discussion is to decide how to compare them. To compare, you need number and the formula. Of course, to make the formula, you have to think much about values because you have to decide what is important so only data is not enough. But data is necessary.

Therefore, two most important things are data analysis and discussion. Data won’t answer what is really important. For instance, data can’t prove that your family is the most important for you although it might help you to realize it. It’s what you have to decide. However, if your priorities are clear, large amount of various data must be helpful to answer what you should do. Data would answer a lot of things but you have to think much about what you ask.
